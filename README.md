# Smart Toy Store Web Application

A modern and interactive e-commerce toy store website built using **HTML, CSS, and JavaScript**.  
The project simulates a real-world toy shopping platform inspired by premium toy retailers like Hamleys, featuring product browsing, category filtering, cart management, user authentication, and checkout experience.

---

# Project Overview

Smart Toy Store is a frontend-based e-commerce application designed to provide customers with a smooth online toy shopping experience.

Users can:

- Browse different toy categories
- Search products
- Apply filters
- Add products to cart
- Manage cart quantities
- Proceed to checkout
- Select payment methods
- Create an account and login

The project focuses on creating a responsive and user-friendly shopping interface using core frontend technologies.

---

# Features

## Product Management

- 50+ toy products
- Multiple categories:
  - Cars & Vehicles
  - Dolls
  - Building Toys
  - Educational Toys
  - Board Games

- Product cards with:
  - Product image
  - Name
  - Price
  - Add to Cart button


## 🔎 Search & Filters

- Real-time product search
- Category-based filtering
- Price range filtering


## Shopping Cart

- Add products to cart
- Increase/decrease quantity
- Remove items
- Automatic price calculation
- Cart persistence using Local Storage


## Authentication System

- User signup
- User login
- Logout functionality
- Local Storage based user session management


## Checkout System

- Checkout interface
- Customer details form
- Payment method selection:
  - UPI
  - Credit Card
  - Debit Card
  - Cash on Delivery


## 🎨 UI Features

- Responsive design
- Modern product grid
- Smooth animations
- Interactive buttons
- Premium e-commerce inspired layout
